Using media queries

Media queries can be applied in three different ways

  • As a link inside the html document in the head tag. Example:
    • <html> <head> <link rel="stylesheet" media="screen and (min-width: 500px)" href="yes.css"> </head> </html>
  • Embed with the @media tag, as it was done in this html document. Example:
    • <html> <head> <style> @media screen and (min-width: 500px) { body { background-color: green; } </style> </head> </html>
  • You can embed then in the main HTML file or the main CSS file with an @import tag.
    • /* Everything 700px and lower */ @import url("./700.css") only screen and (max-width: 700px);
